# 날짜 형식 변경 완료 ✅

## 변경 사항

사용자 요청: "Date 부분 서식을 yyyy-mm-dd로 표시될수있게 해줘"

### 변경 전 vs 변경 후

**변경 전:**
```csv
Date,Customer,...
"Jul. 31, 2025","Daeyoung Chaevi Co., Ltd.",...
"Dec. 22, 2025","Daeyoung Chaevi Co., Ltd.",...
```

**변경 후:**
```csv
Date,Customer,...
2025-07-31,"Daeyoung Chaevi Co., Ltd.",...
2025-12-22,"Daeyoung Chaevi Co., Ltd.",...
```

## 주요 특징

### 1. 표준 ISO 8601 형식
- yyyy-mm-dd 형식 사용
- 국제 표준 날짜 형식
- 데이터베이스 호환

### 2. 다양한 입력 형식 지원
프로그램이 자동으로 인식하는 날짜 형식:
- `Dec. 22, 2025` → `2025-12-22`
- `December. 22, 2025` → `2025-12-22`
- `Jul. 31, 2025` → `2025-07-31`
- `12/22/2025` → `2025-12-22`
- `2025-12-22` → `2025-12-22` (그대로 유지)

### 3. 엑셀에서 사용 장점
- ✅ 정렬이 쉬움 (알파벳순 정렬 = 날짜순 정렬)
- ✅ 필터링이 편함
- ✅ 날짜 계산이 정확함
- ✅ 언어 설정과 무관하게 동일하게 표시됨

### 데스크톱 버전
```bash
python converter.py input.pdf output.csv
```

### Streamlit 웹 앱
1. 웹 브라우저에서 앱 접속
2. PDF 파일 업로드
3. 변환 결과 다운로드
4. 날짜가 자동으로 yyyy-mm-dd 형식으로 변환됨

## 기술 세부사항

### 구현 위치
- 파일: `converter.py`
- 메서드: `format_date_to_iso()`
- 적용 위치: `extract_header_info()`

### 동작 원리
1. PDF에서 날짜 텍스트 추출 (예: "Dec. 22, 2025")
2. `format_date_to_iso()` 메서드 호출
3. 여러 형식 시도하여 파싱
4. yyyy-mm-dd 형식으로 변환
5. CSV 파일에 저장

### 오류 처리
- 날짜 형식을 인식하지 못하는 경우: 원본 텍스트 그대로 저장
- 빈 날짜 필드: 빈 문자열로 처리
- 파싱 실패: 원본 유지 (데이터 손실 방지)
