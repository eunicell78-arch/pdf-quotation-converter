## ❓ 자주 묻는 질문

### Q: 인터넷이 없으면 사용할 수 없나요?
**A:** 처음 웹페이지를 여는 데는 인터넷이 필요합니다. 하지만 한 번 열어두면 인터넷이 끊겨도 변환은 가능합니다 (브라우저를 닫지 않은 경우).

### Q: 파일이 서버로 업로드되나요?
**A:** **절대 아닙니다!** 모든 처리가 여러분의 컴퓨터/기기에서만 이루어집니다. 파일은 서버로 전송되지 않습니다.

### Q: 어떤 브라우저를 사용해야 하나요?
**A:** 대부분의 최신 브라우저에서 작동합니다:
- ✅ Chrome / Edge (추천)
- ✅ Firefox
- ✅ Safari (Mac, iPhone)
- ✅ Samsung Internet (안드로이드)

### Q: 스마트폰에서도 사용할 수 있나요?
**A:** 네! 모든 스마트폰과 태블릿에서 사용 가능합니다. 화면 크기에 맞게 자동으로 조정됩니다.

### Q: 파일 크기 제한이 있나요?
**A:** 브라우저의 메모리에 따라 다릅니다. 일반적인 견적서 PDF (1~10MB)는 문제없이 처리됩니다. 매우 큰 파일(50MB 이상)은 느릴 수 있습니다.

### Q: 변환은 되지만 CSV가 빈 내용으로 나옵니다.
**A**: 
1. **최신 버전 확인**: 브라우저 캐시를 지우고 페이지를 새로고침하세요 (Ctrl+Shift+R 또는 Cmd+Shift+R)
2. **개발자 콘솔 확인**: F12를 눌러 Console 탭에서 오류 메시지를 확인하세요
3. **PDF 형식 확인**: PDF가 올바른 견적서 형식인지 확인하세요
4. **데스크톱 버전 사용**: 복잡한 PDF의 경우 Python 기반 데스크톱 버전이 더 정확합니다

### Q: "PDF에서 데이터를 추출할 수 없습니다" 오류가 나옵니다.
**A:** 다음을 확인해주세요:
1. PDF 파일이 손상되지 않았는지 확인
2. PDF가 올바른 견적서 형식인지 확인
3. 다른 브라우저에서 시도해보기
4. 페이지를 새로고침하고 다시 시도

### Q: 데스크톱 버전과 다른 점이 있나요?
**A:** 기본 기능은 동일합니다. 주요 차이점:
- 웹 버전: 설치 불필요, 모든 기기 지원
- 데스크톱 버전: 오프라인 사용 가능, 약간 더 빠를 수 있음

### Q: 비용이 있나요?
**A:** **완전 무료**입니다! 제한도 없고, 광고도 없습니다.

### Q: 여러 파일을 한 번에 변환할 수 있나요?
**A:** 현재는 한 번에 하나씩 변환해야 합니다. 하지만 변환 후 "새 파일 변환하기" 버튼을 눌러 계속 변환할 수 있습니다.

## 💡 사용 팁

### 1. 즐겨찾기에 추가하세요
웹페이지 주소를 브라우저 즐겨찾기에 추가하면 언제든지 빠르게 접근할 수 있습니다.

### 2. 바탕화면에 바로가기 만들기
#### Chrome/Edge에서:
1. 웹페이지를 엽니다
2. 주소창 옆의 ⋮ (점 3개) 메뉴 클릭
3. "더 많은 도구" → "바로 가기 만들기"
4. "바탕화면에 추가" 체크
5. 이제 바탕화면 아이콘을 더블클릭하면 바로 열립니다!

#### iPhone/iPad에서:
1. Safari에서 웹페이지 열기
2. 공유 버튼 (↑) 탭
3. "홈 화면에 추가" 선택
4. 홈 화면에 아이콘이 생깁니다!

#### Android에서:
1. Chrome에서 웹페이지 열기
2. ⋮ 메뉴 → "홈 화면에 추가"
3. 앱처럼 사용 가능!

### 3. 오프라인 사용 (고급)
브라우저 캐시에 저장되므로, 한 번 열어두면 짧은 시간 동안 인터넷 없이도 사용할 수 있습니다.
