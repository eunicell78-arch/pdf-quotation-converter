# 견적서 변환기 GUI 사용 가이드

## 📌 소개

PDF 견적서 변환기는 PDF 형식의 견적서를 CSV 파일로 자동 변환해주는 프로그램입니다. 
이 가이드는 GUI(그래픽 사용자 인터페이스) 버전의 사용법을 설명합니다.

## 🎯 주요 기능

- ✅ 간편한 GUI 인터페이스
- ✅ PDF 파일 선택 및 자동 변환
- ✅ 변환 진행 상황 표시
- ✅ 자동으로 CSV 파일 저장
- ✅ 변환 완료 후 파일 자동 열기 옵션

## 💻 실행 방법

### 방법 1: BAT 파일로 실행 (Python 설치 필요)

#### 사전 준비
1. Python 3.7 이상 설치 (https://www.python.org/downloads/)
2. Python 설치 시 "Add Python to PATH" 옵션 체크

#### 실행 단계
1. `run_converter.bat` 파일을 바탕화면에 복사
2. `run_converter.bat` 파일을 더블클릭
3. 처음 실행 시 자동으로 필요한 패키지가 설치됩니다
4. GUI 프로그램이 자동으로 실행됩니다

### 방법 2: EXE 파일로 실행 (추천 - Python 설치 불필요)

#### EXE 파일 생성
1. `build_exe.bat` 파일을 더블클릭
2. 빌드가 완료되면 `dist` 폴더에 `견적서변환기.exe` 파일이 생성됩니다
3. `견적서변환기.exe` 파일을 바탕화면에 복사

#### 실행 단계
1. 바탕화면의 `견적서변환기.exe` 파일을 더블클릭
2. 별도의 설치 없이 바로 프로그램이 실행됩니다

## 📖 사용 방법

### 단계별 가이드

#### 1단계: 프로그램 실행
- 바탕화면의 실행 파일 (`.bat` 또는 `.exe`)을 더블클릭합니다
- GUI 창이 열립니다

#### 2단계: PDF 파일 선택
- "1. PDF 파일 선택" 섹션의 **"파일 선택"** 버튼을 클릭
- 변환할 PDF 견적서 파일을 선택
- 선택한 파일 경로가 표시됩니다

#### 3단계: 저장 위치 선택
- "2. 저장 위치 선택" 섹션의 **"저장 위치"** 버튼을 클릭
- CSV 파일을 저장할 위치와 파일명을 지정
- (선택사항) PDF 파일을 선택하면 자동으로 같은 이름의 CSV 파일명이 제안됩니다

#### 4단계: 변환 시작
- **"🚀 변환 시작"** 버튼을 클릭
- 진행 표시줄이 나타나며 변환이 진행됩니다
- 변환이 완료되면 성공 메시지가 표시됩니다

#### 5단계: 결과 확인
- 변환 완료 팝업에서 **"예"**를 클릭하면 CSV 파일이 자동으로 열립니다
- **"아니오"**를 클릭하면 파일은 저장되지만 자동으로 열리지 않습니다

## ❓ 문제 해결

### "Python이 설치되어 있지 않습니다" 오류
- **원인**: Python이 설치되지 않았거나 PATH에 등록되지 않음
- **해결**: 
  1. Python 공식 사이트에서 Python 3.7 이상 다운로드
  2. 설치 시 "Add Python to PATH" 옵션 반드시 체크
  3. 설치 후 컴퓨터 재시작

### "패키지 설치에 실패했습니다" 오류
- **원인**: 인터넷 연결 문제 또는 권한 문제
- **해결**: 
  1. 인터넷 연결 확인
  2. 관리자 권한으로 실행
  3. 수동 설치: 명령 프롬프트에서 `pip install -r requirements.txt` 실행

### "PDF 변환 중 오류가 발생했습니다" 오류
- **원인**: PDF 파일이 올바른 견적서 형식이 아님
- **해결**: 
  1. PDF 파일이 손상되지 않았는지 확인
  2. 올바른 견적서 형식인지 확인
  3. 다른 PDF 파일로 테스트

### GUI 창이 열리지 않음
- **원인**: tkinter 패키지 문제
- **해결**: 
  1. Python 재설치 (tkinter는 기본 포함)
  2. EXE 버전 사용 (방법 2)

### CSV 파일이 열리지 않음
- **원인**: Excel 또는 CSV 뷰어가 설치되지 않음
- **해결**: 
  1. Microsoft Excel 설치
  2. 또는 메모장으로 열기
  3. 또는 온라인 CSV 뷰어 사용

## 💡 유용한 팁

### 1. 바탕화면 바로가기 만들기
- 실행 파일을 바탕화면에 복사하면 더 빠르게 접근할 수 있습니다
- 바로가기 이름을 "견적서 변환"과 같이 알기 쉽게 변경

### 2. 기본 저장 위치
- 프로그램은 바탕화면을 기본 저장 위치로 제안합니다
- 자주 사용하는 폴더가 있다면 해당 폴더로 이동하여 저장

### 3. 파일명 규칙
- PDF 파일명을 기반으로 CSV 파일명이 자동 제안됩니다
- 예: `견적서_2024.pdf` → `견적서_2024.csv`

### 4. 일괄 변환
- 여러 파일을 변환하려면 각 파일마다 변환 과정을 반복하세요
- 각 변환은 독립적으로 진행됩니다

### 5. 파일 백업
- 중요한 견적서는 변환 전 백업을 권장합니다
- 원본 PDF 파일은 변환 과정에서 수정되지 않습니다

## 📊 변환 규칙

프로그램은 다음과 같은 규칙으로 PDF를 CSV로 변환합니다:

| PDF 필드 | CSV 컬럼 | 설명 |
|---------|---------|------|
| To | Customer | 고객사 |
| From | Planner | 담당자 |
| Date | Date | 날짜 |
| Product | Product, Rated Current, Cable Length, Description | 제품 정보 분리 |
| Delivery Term | Delivery Term | 납기 조건 |
| MOQ | MOQ | 최소 주문 수량 |
| Unit Price | Price | 단가 |
| L/T | L/T | 리드타임 |
| Remark | Remark | 비고 |

### 특수 처리
- "Sample" MOQ는 수량 1로 변환
- NRE List 항목은 별도로 처리
- 병합된 셀은 자동으로 반복 입력

## 🔧 고급 사용자를 위한 정보

### 명령줄(CLI) 버전 사용
GUI 대신 명령줄에서 직접 실행하려면:
```bash
python converter.py input.pdf output.csv
```

### 소스 코드 수정
- `converter_gui.py`: GUI 인터페이스 코드
- `converter.py`: 변환 로직 코드
- 수정 후 다시 빌드: `build_exe.bat` 실행

### PyInstaller 옵션 커스터마이징
`build_exe.bat` 파일을 편집하여 다음 옵션을 조정할 수 있습니다:
- `--onefile`: 단일 실행 파일 생성
- `--windowed`: 콘솔 창 숨김
- `--name`: 실행 파일 이름
- `--icon`: 아이콘 파일 지정 (있는 경우)
