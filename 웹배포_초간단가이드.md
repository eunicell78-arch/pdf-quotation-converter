# 🚀 Streamlit 웹 배포 - 초간단 가이드

## ✅ 네, 맞습니다! Streamlit Cloud에 배포하면 웹에서 바로 작동합니다!

배포 후에는 **누구나 URL로 접속**하여 브라우저에서 바로 사용할 수 있습니다.
설치 필요 없음! Python 필요 없음! 그냥 웹 브라우저만 있으면 됩니다!

---

## 📱 3단계로 웹 배포하기 (5분 소요)

### 1단계: Streamlit Cloud 로그인 ⏱️ 1분

1. 브라우저에서 이동: **https://share.streamlit.io**
2. "Sign up" 또는 "Continue with GitHub" 클릭
3. GitHub 계정으로 로그인 허용

![Streamlit Cloud 로그인](https://docs.streamlit.io/images/streamlit-community-cloud/deploy-empty-state.png)

### 2단계: 앱 배포 설정 ⏱️ 2분

1. "New app" 버튼 클릭 (오른쪽 상단)

2. 저장소 정보 입력:
   ```
   Repository: eunicell78-arch/pdf-quotation-converter
   Branch: copilot/add-gui-for-pdf-to-csv
   Main file path: streamlit_app.py
   ```

3. "Deploy!" 버튼 클릭

![App 설정](https://docs.streamlit.io/images/streamlit-community-cloud/deploy-an-app.png)

### 3단계: 완료! ⏱️ 2분 (자동 빌드)

빌드가 완료되면 **자동으로 URL이 생성**됩니다!

```
https://pdf-quotation-converter.streamlit.app
```

또는

```
https://[your-app-name]-[random-id].streamlit.app
```

**이 URL을 누구나 사용할 수 있습니다!** 🎉

```bash

git commit -m "업데이트"
git push
```

→ **Streamlit Cloud가 자동으로 재배포!**

몇 분 후 웹에서 업데이트된 버전 확인 가능! 🎉

## 🐛 배포 중 문제가 생기면?

### 1. 빌드 실패
- Streamlit Cloud 대시보드에서 "Logs" 확인
- requirements.txt 파일 확인
- Python 버전 문제: 3.8-3.11 권장

### 2. 앱이 안 열림
- 몇 분 기다리기 (초기 빌드는 시간 소요)
- 브라우저 새로고침 (Ctrl + F5)
- 다른 브라우저로 시도

### 3. PDF 변환 안 됨
- packages.txt 파일 있는지 확인
- 앱 재시작: Streamlit Cloud에서 "Reboot app"
