## 💻 데스크톱 버전 (5분 소요)

웹 버전이 아닌 컴퓨터에 설치하고 싶으시다면:

## 지금 바로 사용하기! (5분이면 끝!)

### 🎬 따라하기

#### 1️⃣ 다운로드 (1분)
**[👉 이 링크를 클릭하세요!](https://github.com/eunicell78-arch/pdf-quotation-converter/archive/refs/heads/copilot/add-gui-for-pdf-to-csv.zip)**

→ ZIP 파일이 다운로드됩니다
→ ZIP 파일을 **오른쪽 클릭** → **"압축 풀기"**
→ 바탕화면에 압축을 푸세요!

#### 2️⃣ Python 설치 (3분) - 처음 한 번만!

**[👉 Python 다운로드](https://www.python.org/downloads/)**

1. 노란색 다운로드 버튼 클릭
2. 다운로드된 파일 실행
3. ⚠️ **반드시 "Add Python to PATH" 체크!** ⚠️
4. "Install Now" 클릭
5. 완료되면 컴퓨터 재시작

#### 3️⃣ 실행 (1초!)

압축 푼 폴더에서:
- **`run_converter.bat`** 파일 찾기
- 바탕화면으로 복사
- **더블클릭!** 🖱️

#### 4️⃣ 사용 (30초)

프로그램 창이 열리면:
1. "파일 선택" 클릭 → PDF 선택
2. "저장 위치" 클릭 → 저장할 곳 선택
3. "🚀 변환 시작" 클릭
4. 완료! 🎉

## 🆘 문제가 있나요?

### "Python을 찾을 수 없습니다" 오류
→ Python 설치 시 "Add Python to PATH"를 체크 안 하셨습니다
→ Python을 다시 설치하거나, 컴퓨터를 재시작하세요

### 프로그램이 안 열려요
→ Python이 제대로 설치되었는지 확인
→ 관리자 권한으로 실행해보세요 (파일 오른쪽 클릭 → "관리자 권한으로 실행")

### 더 자세한 도움말이 필요해요

→ **[USAGE_GUI.md](USAGE_GUI.md)** 파일에 더 상세한 설명이 있습니다!
