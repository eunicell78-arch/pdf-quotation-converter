## 🌐 웹 버전 - 가장 쉬운 방법! (추천!)

**설치가 필요 없습니다!** 웹브라우저에서 바로 사용하세요.

### 👉 [여기를 클릭하여 웹에서 바로 사용](https://eunicell78-arch.github.io/pdf-quotation-converter/)

#### 웹 버전 특징:
- ✅ **설치 불필요** - Python, 프로그램 설치 전혀 필요 없음
- ✅ **바로 사용** - 링크만 클릭하면 즉시 사용 가능
- ✅ **안전** - 파일이 인터넷으로 전송되지 않음 (컴퓨터에서만 처리)
- ✅ **모든 기기** - 컴퓨터, 노트북, 태블릿, 스마트폰 모두 OK
- ✅ **무료** - 완전 무료, 제한 없음

#### 웹 버전 사용 방법:
1. 위 링크 클릭
2. PDF 파일 선택 또는 드래그
3. "변환 시작" 버튼 클릭
4. CSV 파일 자동 다운로드!

**끝입니다!** 정말 간단하죠? 😊

## 💻 데스크톱 버전 (설치 필요)

웹 버전이 편하지 않으시다면 아래 방법으로 데스크톱 프로그램을 설치할 수 있습니다.

## 🎯 가장 쉬운 방법 (데스크톱)

### 1단계: 파일 다운로드

아래 링크를 클릭하여 필요한 파일들을 다운로드하세요:

👉 **[여기를 클릭하여 파일 다운로드](https://github.com/eunicell78-arch/pdf-quotation-converter/archive/refs/heads/copilot/add-gui-for-pdf-to-csv.zip)**

또는

1. GitHub 페이지로 이동: https://github.com/eunicell78-arch/pdf-quotation-converter
2. 초록색 **"Code"** 버튼 클릭
3. **"Download ZIP"** 클릭
4. 다운로드된 ZIP 파일을 바탕화면에 압축 해제

### 2단계: Python 설치 (처음 한 번만)

Python이 설치되어 있지 않다면:

1. https://www.python.org/downloads/ 접속
2. 큰 노란색 버튼 **"Download Python"** 클릭
3. 다운로드된 파일 실행
4. ⚠️ **중요!** 설치 화면에서 **"Add Python to PATH"** 체크박스를 반드시 체크!
5. "Install Now" 클릭

### 3단계: 프로그램 실행

압축을 푼 폴더에서:

1. **`run_converter.bat`** 파일을 찾습니다
2. 이 파일을 **바탕화면으로 복사**합니다
3. 바탕화면의 **`run_converter.bat`** 파일을 **더블클릭**!
4. 프로그램이 자동으로 실행됩니다! 🎉

---

## 🚀 Python 없이 사용하기 (고급 사용자용)

Python을 설치하기 싫으시다면, 실행 파일(.exe)을 만들 수 있습니다:

### EXE 파일 만들기

1. 위의 1-2단계를 먼저 완료하세요 (Python 설치 필요)
2. **`build_exe.bat`** 파일을 더블클릭
3. 잠시 기다리면 `dist` 폴더가 생성됩니다
4. `dist` 폴더 안의 **`견적서변환기.exe`** 파일을 바탕화면으로 복사
5. 이제 **Python 없이도** 바탕화면의 EXE 파일만 더블클릭하면 됩니다!

## 📱 사용 방법 (아주 간단합니다!)

프로그램을 실행하면 창이 뜹니다:

```
┌─────────────────────────────────────────┐
│   📄 PDF 견적서 → CSV 변환기           │
├─────────────────────────────────────────┤
│                                         │
│  [PDF 파일 선택] 버튼 클릭 👈 여기!     │
│  → PDF 견적서 파일을 선택               │
│                                         │
│  [저장 위치] 버튼 클릭 👈 그 다음!      │
│  → CSV 파일을 저장할 위치 선택          │
│                                         │
│  [🚀 변환 시작] 버튼 클릭 👈 마지막!    │
│  → 변환이 자동으로 진행됩니다           │
│                                         │
└─────────────────────────────────────────┘
```

### 단계별 설명:

1️⃣ **PDF 파일 선택**: 변환하고 싶은 견적서 PDF 파일을 선택합니다

2️⃣ **저장 위치**: CSV 파일을 저장할 곳을 정합니다 (보통 바탕화면이 편합니다)

3️⃣ **변환 시작**: 버튼을 누르면 자동으로 변환됩니다!

4️⃣ **완료**: "파일을 여시겠습니까?"라고 물어보면 **"예"**를 누르세요!

## ❓ 자주 묻는 질문

### Q: ZIP 파일은 어떻게 여나요?
**A:** ZIP 파일을 오른쪽 클릭 → **"압축 풀기"** 또는 **"Extract All"** 선택

### Q: 더블클릭해도 프로그램이 안 열려요!
**A:** 
- Python이 제대로 설치되었는지 확인
- Python 설치 시 "Add Python to PATH"를 체크했는지 확인
- 컴퓨터를 재시작해보세요

### Q: "Python이 없습니다" 오류가 나와요!
**A:** Python을 설치하지 않으셨거나, PATH 설정이 안 된 것입니다. 위의 2단계를 다시 확인하세요.

### Q: EXE 파일은 어디에 있나요?
**A:** `build_exe.bat`를 실행하면 자동으로 `dist` 폴더가 생성되고, 그 안에 `견적서변환기.exe` 파일이 있습니다.

### Q: 더 자세한 설명이 필요해요!
**A:** 같은 폴더에 있는 `USAGE_GUI.md` 파일을 열어보세요!
